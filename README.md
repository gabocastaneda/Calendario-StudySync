# Calendar StudySync

Un calendario hecho en PHP y JavaScript que permite crear eventos, agregando nombre y la duraci√≥n, se pueden mover, eliminar e incluso editar.


## Implementar

Clona el proyecto

```bash
  git clone https://github.com/gabocastaneda/Calendario-StudySync.git
```

Ve al directorio del proyecto

```bash
  cd calendario
```

Crear base de datos en el servidor 

```bash
  Create database 'practicas'
```

Importar el archivo que se encuentra en la carpeta bd

```bash
  eventoscalendar.sql
```
